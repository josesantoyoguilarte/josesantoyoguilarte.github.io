Featured personal project

MixReady

An ASP.NET Core 8 web app that turns any song into a DJ-ready track.

MixReady uses a hybrid C# (NAudio) and Python (librosa + Demucs) pipeline to detect BPM, key, and song structure, separate stems, and synthesize beat-aligned intros that crossfade smoothly into the original mix — mimicking the workflow a professional DJ uses to prepare tracks for a set.

Highlights

  • Smart intro generation with beat-aligned crossfades into the original track.
  • AI stem separation (drums, bass, vocals, other) with per-stem intro selection.
  • Interactive WaveSurfer.js waveform — click to pick extraction and song-start points, snapped server-side to the beat grid.
  • Auto-detection of BPM, musical key, and genre on upload.
  • Pitch and tempo shifting without artifacts.

Architecture

  • Clean separation between Razor Pages / Web API, domain services, audio helpers, and background jobs.
  • Resilient pipeline — every Python-powered feature has a pure-C# fallback (BPM, key, genre, drum isolation/patterns).
  • Redis-backed track metadata and async job state (intro generation, stem separation).
  • Containerized with Docker / docker-compose for a QA environment.
